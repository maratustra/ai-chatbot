# AI Chatbot

Pet project of an interactive AI chatbot.

## Technologies

- React 19
- TypeScript
- Vite
- Tailwind CSS
- pnpm

## Installation and Setup

### Install dependencies:
```bash
pnpm install
```

### Run dev server:
```bash
pnpm dev
```
