# AI Chatbot

Пет-проект интерактивного чат-бота с AI.

## Технологии

- React 19
- TypeScript
- Vite
- Tailwind CSS

## Установка и запуск
# Установка зависимостей
npm install

# Запуск dev-сервера
npm run dev

## Функциональность

будет вноситься по мере добавления

Проект находится в разработке в рамках [DevChallenges.io](https://devchallenges.io/challenge/ai-chat-bot)
